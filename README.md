# Arduino Frontend ES

This is an example ReadMe with light selections.

## Run Locally

Clone the project

```bash
  git clone
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
npm install
```

or using yarn

```bash
yarn install
```

Start the server

```bash
npm run start
```

```bash
yarn start
```
